# **App Name**: Kiss Day Dreams

## Core Features:

- Animated Gradient Background: Display a continuously animated gradient background with floating hearts.
- Heart Trail Effect: Add a mouse heart trail effect for added engagement.
- Secret Message: Opens an animated popup with romantic text on button click.
- Send Virtual Kiss: Display a large animated kiss with floating sparkles on button click.
- Love Meter: Show an animated love percentage (random 85%-100%) with heartbeat animation on button click.
- Theme Switching: Allow users to toggle between 'Light Pink Romance', 'Dark Red Passion', 'Dreamy Purple Glow', and 'Night Romantic' themes, saved in localStorage.
- Surprise Animation: Trigger a full-screen heart explosion animation with a romantic message reveal on button click.

## Style Guidelines:

- Primary color: Soft Pink (#FFC0CB) to evoke romance and tenderness.
- Background color: Very light desaturated pink (#F8E8EE) for a dreamy feel.
- Accent color: Lavender (#E6E6FA) to complement the pink with a gentle contrast.
- Body and headline font: 'Belleza', a humanist sans-serif with personality for headlines, paired with 'Alegreya', a humanist serif for longer text.
- Employ glassmorphism for central cards, and modern UI principles, alongside soft shadows.
- Utilize smooth transitions for all interactive elements.
- Implement heartbeat pulse animation on the main title, and button hover glow + bounce animation for a premium feel.
